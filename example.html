backbrace.js

A simple validation extension for backbone.js and jquery.

Instead of approaching the problem of client-side validation from the DOM/view side, backbrace takes the same stance as backbone, implementing validation in the existing backbone models. It passes events down to the validator using backbone's event system, and allows you to extend a new kind of backbone object, Backbone.Form, which itself extends Backbone.View.
